# Story Hunting — History Narrative

Mandatory, tool-driven enrichment. Stories make the piece sticky; **facts still come from the lesson pack**.

## Rules

- Hunt **after** ingesting the lesson pack, not instead of it.  
- Only keep stories that **illustrate a claim already supported** (or clearly marked contested) in the pack.  
- Prefer named people, dated moments, sensory places.  
- Verify dates/names lightly; if the pack and a secondary source conflict, prefer the pack and label uncertainty.  
- Collect 3–5 candidates; use the best **1–2** (or one per major chapter in a long mini-book).  
- Encyclopedia background: **Grokipedia** — not Wikipedia.  
- Do not invent quotes; short primary lines must be real and attributed.

## Search strategies (use all that apply)

1. **The person under pressure** — ruler, prophet, soldier, refugee, chronicler at a decision moment.  
2. **The failure before the fall** — ignored warning, broken alliance, failed revolt.  
3. **The ordinary witness** — siege hunger, exile road, temple vessel carried off (when sources allow).  
4. **The document that still speaks** — chronicle line, lament verse, inscription, treaty clause.  
5. **The later memory** — how a later age commemorated the event (fast day, monument, national myth) — labeled as memory when it is not the event itself.  
6. **The rival’s view** — what the other empire’s records claim.  
7. **The cross-domain parallel** (rare) — only if it clarifies mechanism without anachronism.

## Execution

- Use web search and, when useful, X search for recent public discussion of primary sources or museum finds — never as sole proof of ancient fact.  
- Browse promising pages for names, dates, places, dialogue, sensory detail, motives.  
- Log story sources in narrative `sources.md`.

## What not to hunt for

- Gratuitous gore or trauma detail beyond what the profile and Core Message need  
- Nationalist fanfic that the lesson pack already rejected  
- “Fun facts” that do not serve the spine  

## Link to visuals

When a hunted story becomes a chapter spine, ensure an illustration exists: reuse a lesson asset if it fits; otherwise delegate a **story illustration** worker (period image or labeled modern reconstruction) — same honesty rules as the history skills.
